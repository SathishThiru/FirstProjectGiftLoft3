This project has 3 classes as follows:
1. NavigateGiftLoft
2. ExcelRead
3. ExcelWrite

And it has Screenshots folder to record alll the screenshots at important stages during the product search and checkout

And also it has 2 Excel files as follows:
1. ExcelInputGivingFile - this File contains login credentials
2. ExcelOutputRecordingFile - this file records the name and price of the item which is fetched from the final checkout page

Note:
This project uses Maven
